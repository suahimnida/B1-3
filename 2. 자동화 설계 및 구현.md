# 채용공고 자동 저장

1. 채용공고 자동 저장을 주제로한 자동화 워크플로우 구현
2. trigger, action, 조건분기(필터/라우터) 1개 이상씩 포함

문서
1. 도구 및 도구 선정 이유
2. 워크플로우 설계 문서 (설명 또는 다이어그램)
3. 워크플로우 흐름 설명 포함)
4. 주제 선정 이유
5. 구현 화면
6. 자동 실행 구조 구현
7. 실행 결과 화면 캡쳐
8. 포함한 트리거, 액션, 조건분기(필터/라우터) 기재
   (조건 분기 구조가 포함된 경우, 각 분기 경로가 실제로 1회 이상 실행된 결과를 확인할 수 있어야 함)
9. API Key, 토큰, 비밀버호 등 민감정보 노출X (필요한 값은 마스킹 처리)
10. 계정/연동 환경 리스크 완화 가이드 (구글 시트 + 지메일 + 웹훅 or 구글 시트 + 슬랙or디스코드 + 웹혹)
11. 트리거 발생 시, 자동 실행되도록 설정
